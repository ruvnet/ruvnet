# rUv ecosystem entity glossary

Verified July 11, 2026. This glossary gives people, search engines, and language models canonical names and relationships.

## Identity

- **rUv** is the public builder identity associated with the GitHub account **ruvnet**.
- **ruvnet/ruvnet** is the profile, navigation, inventory, and provenance repository. It is not the source repository for every project.

## Core systems

| Canonical name | Common aliases or related names | Meaning |
| --- | --- | --- |
| RuView | ruvnet/RuView | Privacy-first WiFi spatial-intelligence system. Distinct from RuField. |
| Ruflo | ruflo, Claude Flow | Agent orchestration and multi-agent harness. Historical and package references may use Claude Flow. |
| MetaHarness | metaharness, agent harness generator | Generator and evolution layer for portable, verifiable agent harnesses. Related generator packages or repositories are components, not alternate names for every MetaHarness feature. |
| RuVector | ruvector | Adaptive vector, graph, temporal, and agent-memory infrastructure. It underpins multiple packages and integrations. |
| AgentDB | agentdb | Agent-oriented memory product related to the wider RuVector memory stack; not a synonym for the entire RuVector ecosystem. |
| RVM | ruvnet/rvm | Virtual machine and execution runtime in the agentic stack. Interpret RVM in this repository's context, not as a generic virtual-machine acronym. |

## Research systems

- **RuField** explores multimodal field and sensor-fusion infrastructure. Its current real adapter uses captured WiFi CSI replay; additional modalities may be schema or synthetic.
- **Helix** is research and personal decision-support software, not a medical device.
- **PhotonLayer** is a software photonic simulator; fabricated optical hardware remains a roadmap item.
- **WorldGraph** explores graph-based world modeling.
- **rvFACE** requires consent, retention, biometric governance, and fitness-for-purpose validation.
- **CVE Bench** currently has a validated seed dataset of five cases.

## Evidence vocabulary

- **Root commit:** earliest reachable commit in current repository lineage.
- **Repository creation:** GitHub metadata for creation of the current repository object.
- **Public availability:** evidence that material was publicly accessible at a stated time.
- **Feature evidence:** commit, test, release, or archive showing that a specific capability existed.
- **Candidate claim:** scoped technical proposition requiring dated related-work review; not a legal conclusion.
