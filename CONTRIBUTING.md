# Contributing

This repository is the navigation, inventory, and provenance layer for the rUv open-source ecosystem.

## Useful contributions

- Correct a broken project, package, release, or evidence link.
- Report a stale metric with its source and observation date.
- Clarify a project relationship or alias.
- Improve accessibility, navigation, or factual wording.
- Supply feature-level evidence for a dated technical claim.

## Pull requests

Keep changes narrowly scoped. For factual updates, include the authoritative source, exact observation date, and counting rule. Distinguish repository creation, commit authorship, public availability, release publication, feature presence, adoption, and novelty.

Do not convert scoped candidate claims into broad novelty or legal conclusions. Individual projects have their own contribution and licensing rules.
